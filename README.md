# marcschraepler.github.io

Minimal personal landing page for [Marc Schraepler von Gerlach](https://marcschraepler.com/en/) – Digital Transformation Consultant.

## Overview

A lightweight, single-file landing page in the style of about.me. No build tools, no dependencies, no JavaScript frameworks. Just HTML + inline CSS with Google Fonts.

**Live:** [marcschraepler.github.io](https://marcschraepler.github.io/)

## Features

- Single `index.html` (< 4 KB)
- Responsive design
- Automatic dark mode via `prefers-color-scheme`
- Reduced motion support via `prefers-reduced-motion`
- Zero JavaScript dependencies
- Hosted via GitHub Pages

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | Semantic HTML5 |
| Styling | Inline CSS with custom properties |
| Fonts | Bricolage Grotesque, Plus Jakarta Sans (Google Fonts) |
| Hosting | GitHub Pages |
| Assets | Profile image served from [marcschraepler.com](https://marcschraepler.com) |

## Local Development

No build step required. Open `index.html` in any browser.

```bash
open index.html
```
